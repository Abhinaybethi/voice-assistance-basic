## 🚀 Overview

Neural OS is an AI-powered desktop assistant designed to function as an intelligent operating system rather than a traditional chatbot.

It combines voice interaction, AI reasoning, live telemetry, system automation, web intelligence, and modular AI agents into a futuristic control center.

The goal is to create an assistant capable of understanding natural language, executing system-level tasks, retrieving real-time information, and continuously expanding through plug-in agents.

## ⚙ Installation

Clone the repository

```bash
git clone https://github.com/Abhinaybethi/voice-assistance-basic.git
```

Move into the project

```bash
cd voice-assistance-basic
```

Install frontend

```bash
npm install
```

Install backend

```bash
pip install -r requirements.txt
```

Start backend

```bash
uvicorn main:app --reload
```

Start frontend

```bash
npm run dev
```
